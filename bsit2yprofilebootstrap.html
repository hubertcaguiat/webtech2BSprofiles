BSIT Profiles in Bootstrap Framework

This web page showcases the profiles of BSIT 2Y students, creatively designed with Bootstrap, a popular front-end framework for building responsive and visually appealing websites. The layout dynamically adjusts, displaying six columns for desktop users, three for tablets, and gracefully condenses to a single column for mobile devices. Each student's card features a distinctive image alongside their name. This interactive presentation provides an engaging visual overview of the students, facilitating easy navigation and a seamless user experience across various screen sizes.
